# Sērēnum

An AI skincare advisor that translates live weather data — UV index, humidity, and air quality — into personalised skin advice.

Built with LangChain and Chainlit, using the Anthropic API for reasoning, OpenWeatherMap for live conditions, and Tavily for evidence-backed myth-busting.
